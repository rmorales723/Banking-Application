# Banking-Application

This simple banking application greets the user and asks them which function they would like to perform. Options are:
####
A. Check balance 
####
B. Make deposit
####
C. Make withdrawal 
####
D. View previous transaction
####
E. Calculate interest
####
F. Exit the application
####
